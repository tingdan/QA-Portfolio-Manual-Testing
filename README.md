## 🚀 Project 1: SauceDemo E-Commerce Testing
**Goal:** End-to-end functional testing of a standard e-commerce journey.

### 📋 What I Tested:
- **Login:** Positive (Standard user) & Negative (Locked out user).
- **Cart Logic:** Adding/Removing items and verifying badge counts.
- **Checkout:** Validation of mandatory user information fields.

### 🐛 Key Bugs Found:
1. **[UI] Broken Product Images:** Logged in as `problem_user`, all thumbnails fail to load unique assets.
2. **[Validation] Empty Postal Code:** System allows progression with specific blank fields (Potential data integrity issue).

### 🛠️ Tools Used:
- **Test Management:** Google Sheets / Excel
- **Defect Tracking:** Markdown & Screenshots
- **Browser:** Chrome DevTools
